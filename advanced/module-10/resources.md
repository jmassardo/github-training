## Resources & Further Learning

### Official Documentation

| Resource | Description |
|----------|-------------|
| [REST API Documentation](https://docs.github.com/en/rest) | Complete REST API reference |
| [GraphQL API Documentation](https://docs.github.com/en/graphql) | GraphQL schema and guides |
| [Webhooks Documentation](https://docs.github.com/en/webhooks) | Webhook configuration and events |
| [GitHub Apps Guide](https://docs.github.com/en/apps) | Building and managing GitHub Apps |
| [GraphQL Explorer](https://docs.github.com/en/graphql/overview/explorer) | Interactive GraphQL IDE |

### Tools

| Tool | Purpose |
|------|---------|
| [GitHub CLI (gh)](https://cli.github.com/) | Command-line API access |
| [Octokit](https://github.com/octokit) | Official API client libraries |
| [Probot](https://probot.github.io/) | Framework for building GitHub Apps |
| [Smee.io](https://smee.io/) | Webhook debugging and forwarding |

- [Probot Settings App](https://github.com/probot/settings) - Repository settings as code
- [Delete Merged Branch](https://github.com/SvanBoxel/delete-merged-branch) - Auto-delete branches
- [Auto-assign](https://github.com/kentaro-m/auto-assign) - Auto-assign PR reviewers
