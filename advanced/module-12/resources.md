## Quick Reference

### Audit Log Event Categories

```
repo.*           - Repository operations
org.*            - Organization settings
team.*           - Team management
protected_branch.* - Branch protection
integration.*    - App and OAuth events
business.*       - Enterprise settings
user.*           - User account events

```

### Common Audit Log Queries

```bash
# Repository visibility changes
gh api /orgs/ORG/audit-log?phrase=action:repo.access

# Admin permission grants
gh api /orgs/ORG/audit-log?phrase=action:repo.add_member+permission:admin

# Branch protection modifications
gh api /orgs/ORG/audit-log?phrase=action:protected_branch

# Repository deletions
gh api /orgs/ORG/audit-log?phrase=action:repo.destroy

```

### Useful GraphQL Queries

```graphql
# Get repository security settings
query {
  repository(owner: "org", name: "repo") {
    hasVulnerabilityAlertsEnabled
    isSecurityPolicyEnabled
    defaultBranchRef {
      branchProtectionRule {
        requiredApprovingReviewCount
        requiresCodeOwnerReviews
      }
    }
  }
}

```
