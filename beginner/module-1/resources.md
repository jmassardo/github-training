## Glossary

Key terms from this module:

| Term | Definition |
|------|------------|
| **SDLC** | Software Development Life Cycle - structured phases for building software |
| **Waterfall** | Sequential methodology where phases complete before the next begins |
| **Agile** | Iterative methodology with short cycles and continuous feedback |
| **DevOps** | Culture combining development and operations for faster delivery |
| **CI/CD** | Continuous Integration / Continuous Deployment - automation practices |
| **Shift Left** | Moving testing and security earlier in the development process |
| **Lead Time** | Time from code commit to production deployment |
| **DORA Metrics** | Four key metrics for measuring DevOps performance |
| **InnerSource** | Applying open source practices within an organization |
| **ADR** | Architecture Decision Record - documenting technical decisions |
