# GitHub Training Program for CSMs & CSAs

**Welcome to the complete, hands-on GitHub training program!** This comprehensive curriculum is designed to take you from GitHub novice to expert advisor, following the natural software development lifecycle.

## üéØ Who This Is For

- **Experienced CSMs** transitioning into GitHub roles
- **Customer Success Architects** new to developer tools
- **Technical Account Managers** entering the DevOps space
- Anyone with strong customer-facing skills but no prior GitHub/software development experience

## üìö Program Structure

This 12-week program is organized into **three progressive phases**, each containing multiple focused modules that build upon each other:

## üéì Learning Approach

Each module follows a consistent structure designed for hands-on learning:

### 1. **Context & Overview** 
Understand *why* this topic matters and where it fits in the SDLC

### 2. **Core Concepts**
Learn the fundamental concepts and terminology with clear explanations

### 3. **Guided Walkthrough**
Step-by-step instructions with screenshots and commands

### 4. **Hands-On Lab**
Apply what you learned in a practical exercise

### 5. **Knowledge Check**
Test your understanding with quiz questions

### 6. **Resources**
Links to official docs, Microsoft Learn modules, and GitHub Skills courses

### 7. **Real-World Scenarios**
Customer use cases and common questions you'll encounter as a CSM/CSA

## üèÖ Assessment & Certification

### Continuous Assessment
- **Weekly Lab Completion** (50%) - Hands-on exercises with working repositories
- **Knowledge Checks** (25%) - End-of-module quizzes and demonstrations
- **Phase Milestones** (25%) - Three major projects demonstrating cumulative skills

### Phase Milestones
- **Beginner Milestone:** Present a repository with proper branching, protection rules, and collaboration workflow
- **Intermediate Milestone:** Build and demo a complete CI/CD pipeline with security scanning and DORA metrics
- **Advanced Milestone:** Conduct an enterprise security audit with governance recommendations

### GitHub Certifications
